# 字符字生成器

一个字符生成字符字工具，你搁这搁这呢？🌚


 - 支持 11px 和 7px 两种字符分辨度
 - 支持两倍宽（方便用于行高较高的平台）



## 项目地址
[https://lab.magiconch.com/fontfont/](https://lab.magiconch.com/fontfont/?from=github)


## GitHub
[https://github.com/itorr/fontfont](https://github.com/itorr/fontfont)

## 利用
 - 11px 使用 最像素 Zpix 字体库
 - 7px 使用 观致 8px 字体库
